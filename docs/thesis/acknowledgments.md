# กิตติกรรมประกาศ (Acknowledgments)

---

การค้นคว้าอิสระฉบับนี้สำเร็จลุล่วงได้ด้วยความกรุณาและการสนับสนุนจากหลายฝ่าย ผู้วิจัยขอกราบขอบพระคุณ **ผู้ช่วยศาสตราจารย์ ดร.ปริญญา สุวรรณศรีคำ** อาจารย์ที่ปรึกษาการค้นคว้าอิสระ ผู้ให้คำแนะนำ ชี้แนะแนวทาง และให้กำลังใจตลอดระยะเวลาการทำวิจัย

ขอขอบพระคุณ **คณาจารย์ประจำหลักสูตรวิทยาศาสตรมหาบัณฑิต สาขาวิชาวิศวกรรมซอฟต์แวร์ วิทยาลัยศิลปะ สื่อ และเทคโนโลยี มหาวิทยาลัยเชียงใหม่** ทุกท่านที่ได้ประสิทธิ์ประสาทวิชาความรู้ อันเป็นรากฐานสำคัญในการดำเนินงานวิจัยครั้งนี้

ขอขอบคุณ **อาจารย์ผู้เชี่ยวชาญด้านมวยไท้เก๊กสกุลเฉิน** ที่กรุณาให้ความรู้เกี่ยวกับหลักการของท่าม้วนไหม และสาธิตท่ารำเพื่อใช้เป็นข้อมูลอ้างอิงในการพัฒนาระบบ

ขอขอบคุณ **กลุ่มผู้เข้าร่วมการทดสอบระบบ** ทุกท่านที่สละเวลาอันมีค่าในการทดลองใช้งานและให้ข้อเสนอแนะที่เป็นประโยชน์ต่อการปรับปรุงระบบ

ขอขอบคุณ **เพื่อนร่วมรุ่น** ที่คอยให้กำลังใจและแลกเปลี่ยนความรู้ตลอดระยะเวลาการศึกษา

สุดท้ายนี้ ขอกราบขอบพระคุณ **ครอบครัว** ที่เป็นกำลังใจสำคัญและสนับสนุนทุกอย่างตลอดมา
