## ฉ.1 วัตถุประสงค์การทดสอบ

การทดสอบความสามารถใช้งาน# Appendix G: User Testing Results) มีวัตถุประสงค์เพื่อ:

1. ประเมินความง่ายในการใช้งานระบบ
2. วัดประสิทธิผลของ Real-time Feedback
3. รวบรวมข้อเสนอแนะสำหรับการปรับปรุง

## ฉ.2 กลุ่มผู้ทดสอบ

### ฉ.2.1 ขนาดกลุ่มตัวอย่าง

| รายละเอียด | จำนวน |
|------------|:-----:|
| **จำนวนผู้ทดสอบ** | 8 คน |
| **วิธีการคัดเลือก** | Purposive Sampling |

### ฉ.2.2 เกณฑ์การคัดเลือก (Inclusion Criteria)

- อายุ 18 ปีขึ้นไป และยินยอมเข้าร่วมการทดสอบ
- มีความสนใจในการใช้เทคโนโลยีเพื่อช่วยฝึกซ้อมด้วยตนเอง
- สนใจหรือเคยฝึกฝนมวยไท้เก๊กสกุลเฉินระดับเริ่มต้น-ระดับกลาง
- สามารถเข้าถึง Desktop/Laptop ที่มี Webcam ได้

### ฉ.2.3 เกณฑ์การคัดออก (Exclusion Criteria)

- ไม่เป็นผู้ฝึกระดับสูง / ผู้เชี่ยวชาญ / ครูผู้สอนมวยไท้เก๊ก
- ไม่มีประสบการณ์ในการฝึกมวยไท้เก๊กสกุลเฉินมาก่อน

### ฉ.2.4 คุณสมบัติผู้ทดสอบ

| เกณฑ์ | การกระจาย |
|-------|-----------|
| **ช่วงอายุ** | 20-60 ปี |
| **ประสบการณ์ไท่จี๋** | ไม่มี - มากกว่า 2 ปี |
| **ความสามารถทางเทคโนโลยี** | พื้นฐาน - ขั้นสูง |

## ฉ.5 ข้อสรุปและข้อเสนอแนะ

### ฉ.5.1 จุดแข็งของระบบ

1. **Real-time Feedback ได้รับความพึงพอใจสูงสุด** (75%)
2. ผู้ใช้สามารถ **รู้ว่าทำผิดตรงไหน** ได้ดี (4.4/5.0)
3. **ความเร็วในการตอบสนอง** เหมาะสม (4.3/5.0)

### ฉ.5.2 จุดที่ควรปรับปรุง

1. เพิ่มวิดีโอสาธิตท่าก่อนเริ่มฝึก
2. ขยายจำนวนท่าฝึกให้หลากหลายขึ้น
3. เพิ่มระบบบันทึกประวัติการฝึก

### ฉ.5.3 แผนการพัฒนาในอนาคต

| ลำดับ | รายการ | Phase |
|:----:|--------|:-----:|
| 1 | เพิ่มท่าม้วนไหมสองมือ | Phase 2 |
| 2 | ระบบผู้ใช้และประวัติการฝึก | Phase 2 |
| 3 | ML/DL Model สำหรับประเมินคุณภาพ | Phase 3 |
